#####  ==========  switching betweenn networkmanager  <--->  wpa+dhcpd :
- hier networkmanager is using dhcpcd ! inserted in its config-file! see below!)
- if want persistent-switch to NWmgm /OR wpa+dhcp, then do also:  + systemctl enable/disable  !!

_______:  switch-to networkmanager:
systemctl   stop  dhcpcd.service  wpa_supplicant.service      #--or also "disable" if want persistent
systemctl  restart  NetworkManager      #--or also "enable" if want persistent

_______:  switch-to wpa+dhcpd :
systemctl  stop  NetworkManager         #--or also "disable" if want persistent
systemctl   restart  dhcpcd.service  wpa_supplicant.service      #--or also "enable" if want persistent

-- https://www.reddit.com/r/archlinux/comments/1abq6bp/internet_wifi_connection_involving_networkmanager/   :  NW-manager <--> dhcpcd + wpa_supplicant 
	You need to either:
	a) stick with your original setup of dhcpcd + wpa_supplicant
	b) or, if you really want to use NetworkManager, you disable dhcpcd and wpa_supplicant, now, NetworkManager uses wpa_supplicant for the wifi but it shouldn't need to be enabled by the user himself.

	systemd-/service-files in:        /etc/NetworkManager/NetworkManager.conf  +  /etc/NetworkManager/conf.d/
	profiles/WLAN-config-files in:    /etc/NetworkManager/system-connections/
	runtimes in:		/var/run/NetworkManager/

	- Additional configuration files can be placed in /etc/NetworkManager/conf.d/ : Usually no configuration needs to be done to the global     defaults.
